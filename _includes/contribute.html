O que é o DesenvolvimentoAgil.com.br?

O DesenvolvimentoAgil.com.br é um projeto Open Source feito pela comunidade de desenvolvimento ágil do Brasil. Mudanças e adições tanto no conteúdo quando no layout são bem-vindas, desde que sigam o Guideline de contribuição para o projeto.

Como contribuir?

Se você for desenvolvedor, leia o README do projeto. Senão, solicite mudanças seguindo os passos:

  • Crie uma conta no GitHub, caso não possua.
  • Certifique-se de que não exista nenhuma issue igual a sua.
  • Crie uma issue com a sua solicitação.
  • Descreva claramente a mudança que você deseja fazer.
